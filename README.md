# Мій власний мікросервісний проєкт

Це репозиторій для навчального проєкту в межах курсу "DevOps CI/CD".

У цьому проєкті планується розробка мікросервісної архітектури з використанням сучасних практик DevOps:
- керування версіями через Git та GitHub;
- автоматизація збірки й перевірок через CI/CD;
- контейнеризація сервісів;
- підготовка до розгортання в хмарному або локальному середовищі.

Репозиторій буде поступово доповнюватися кодом сервісів, конфігураціями інфраструктури та автоматизованими пайплайнами.
